Agora uma rotina para testar se é um triângulo retângulo:

  1. Criar uma função que calcule a distância das coordenadas
  2. Calcule as distâncias entre os 3 vértices e armazena em uma lista.
  3. Calcula a maior distância, essa vai ser a candidata à hipotenusa.
  4. Retira da lista com as distâncias, essa nova lista terão os catetos.
  5. Testa se a hipotenusa e os catetos estão de acordo com o teorema de Pitágoras (a2=b2+c2a^{2}=b^{2}+c^{2}a2=b2+c2):
#Cria a função
def distancia(x,y):
    return sqrt((x[0]-y[0])^2+(x[1]-y[1])^2)

#Calcula as distâncias
dAB=distancia(a,b)
dAC=distancia(a,c)
dBC=distancia(b,c)

#Armazena numa lista
distancias=[dAB,dAC,dBC]
#Calcula a maior distância que será a hipotenusa
hipotenusa=max(distancias)
#Remove a hipotenusa, sobram os catetos
distancias.remove(hipotenusa)
catetos=distancias

if (hipotenusa^2==catetos[0]^2+catetos[1]^2):
    print("O triângulo ABC é um triângulo retângulo")
else:
    print("Não é um triângulo retângulo")

def area(poligono):
    vertices=len(poligono)
    lado=distancia(poligono[0],poligono[1])
    semiperimetro=lado*vertices/2
    apotema=(lado/2)/tan(pi/vertices)
    return semiperimetro*apotema

areas=[round(area(poligono_A),3),round(area(poligono_B),3),round(area(poligono_C),3)]
